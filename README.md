# 04 Web APIs: Code Quiz

Welcome to the **Code Quiz** app! This application is designed to help you practice and test your JavaScript fundamentals while simulating the experience of a timed coding assessment. Whether you're preparing for a coding interview or just want to gauge your progress, this quiz will challenge your knowledge and skills.

## Description

As you embark on your journey to becoming a full-stack web developer, it's important to be prepared for coding assessments that may be part of your interview process. This Code Quiz app replicates the format of a typical coding assessment, featuring both multiple-choice questions and interactive coding challenges. The app runs in the browser and is powered by your JavaScript code, providing dynamically updated HTML and CSS for a polished and responsive user interface.

## User Story

As a coding boot camp student,
I want to take a timed quiz on JavaScript fundamentals that stores high scores,
So that I can gauge my progress compared to my peers.

## Acceptance Criteria

To successfully complete this challenge, you must meet the following criteria:

1. When the user clicks the start button, a timer starts, and a question is presented.
2. After answering a question, the user is presented with another question.
3. If a question is answered incorrectly, time is subtracted from the clock.
4. The game is over either when all questions are answered or when the timer reaches 0.
5. Once the game is over, the user can save their initials and score.

## Features

- Timed quiz with multiple-choice questions
- Interactive and dynamic user interface
- High score tracking
- Responsive design for various screen sizes

## Usage

To use the **Code Quiz** app, follow these steps:

1. Open the app in a web browser.
2. Click the "Start" button to begin the quiz.
3. Answer each question to the best of your ability.
4. If you answer a question incorrectly, time will be subtracted from the clock.
5. Once you've completed all questions or the timer reaches 0, the game will be over.
6. Enter your initials to save your score on the high score board.
